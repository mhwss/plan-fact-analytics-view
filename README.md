# Plan–Fact Analytics View
Портфолио-версия SQL-витрины данных для план-факт аналитики.
Проект демонстрирует подход к агрегированию и объединению плановых и фактических показателей для использования в BI-дашбордах и регулярной отчетности.
## Описание
Витрина агрегирует данные по:
- месяцам
- менеджерам
- группам продуктов
И объединяет плановые и фактические значения в единую структуру, удобную для анализа выполнения планов и расчёта отклонений.
Проект выполнен в упрощённом виде и отражает архитектурный подход к построению аналитических витрин данных.
## Реализовано
- агрегация фактических данных
- агрегация плановых показателей
- объединение плана и факта в одной витрине
- расчёт отклонений по количеству и сумме
- подготовка данных для BI-инструментов
## Технологии
- SQL (PostgreSQL)
- CTE (WITH)
- агрегатные функции
- VIEW
## Использование
Витрина может использоваться как источник данных для:
- BI-дашбордов
- план-факт отчетности
- мониторинга выполнения планов
- анализа отклонений
## Примечание
Проект является упрощённой портфолио-версией рабочей аналитической витрины.
Все названия схем и таблиц обезличены и используются исключительно в демонстрационных целях.
